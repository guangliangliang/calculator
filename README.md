# 行业计算器合集 - UniApp

基于 UniApp 开发的多行业计算器项目，适配微信小程序、H5、支付宝小程序等平台。

## 功能概览

当前共 `3` 个行业分类、`30` 个计算器：

### 1. 餐饮/零售行业 - 13 个

- 毛利率计算器
- 菜品定价计算器
- 库存周转计算器
- 促销折扣计算器
- 房租占比计算器
- 人力成本计算器
- 会员折扣计算器
- 进货批量计算器
- 盈亏平衡点计算器
- 翻台率计算器
- 客单价计算器
- 外卖到手收入计算器
- 门店回本周期计算器

### 2. 建筑/装修行业 - 10 个

- 瓷砖用量计算器
- 地板用量计算器
- 涂料用量计算器
- 面积体积计算器
- 水电改造预算
- 工期计算器
- 材料总价计算器
- 装修预算计算器
- 水泥沙子砖用量计算器
- 电线线管用量计算器

### 3. 银行/金融行业 - 7 个

- 房贷计算器
- 车贷计算器
- 存款利息计算器
- 信用卡分期计算器
- 投资收益计算器
- 个人所得税计算器
- 提前还贷计算器

## 技术特点

- 配置驱动架构：所有计算器通过配置定义，便于扩展
- 通用计算页面：大部分新增公式无需单独开发页面
- 本地历史记录：自动保存计算历史
- 结果组件可扩展：复杂场景支持独立结果面板
- 提供校验脚本：可快速验证公式和配置完整性

## 项目结构

```text
calculator/
├── pages/
│   ├── index/                # 首页
│   ├── industry-list/        # 行业列表页
│   ├── calculator/           # 通用计算器页面
│   └── schedule-detail/      # 还款明细页
├── utils/
│   ├── calculators/          # 各行业计算器配置
│   │   ├── catering.js
│   │   ├── construction.js
│   │   └── finance.js
│   ├── calculator-config.js  # 配置中心
│   ├── calculator-form.js    # 表单处理
│   ├── formatter.js          # 格式化与存储
│   └── xlsx-export.js        # 明细导出
├── scripts/
│   └── verify-calculators.js # 计算器校验脚本
├── static/
└── pages.json
```

## 本地开发

```bash
npm install
npm run dev:h5
```

也可以使用 HBuilderX 直接打开项目运行到对应平台。

## 校验脚本

```bash
npm run test:calculators
```

该脚本会检查：

- 计算器 `id` 是否唯一
- 输入/输出配置是否完整
- 核心公式是否返回预期结果
